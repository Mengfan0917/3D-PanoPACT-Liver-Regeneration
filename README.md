# 3D-PanoPACT: Hepatic Vascular Imaging Code

Code for: Hepatic Vascular Remodeling and Metabolic Recovery 
During Liver Regeneration: Longitudinal Photoacoustic Computed 
Tomography in Mice (Submitted to Radiology, 2025)

## Requirements
- MATLAB R2025a
- Image Processing Toolbox
- Parallel Computing Toolbox
- NVIDIA GPU (CUDA support)

## File Description
1_reconstruction_3D_PanoPACT.m    - 3D image reconstruction
2_segmentation_3D_vascular.m      - Hepatic region segmentation  
3_vascular_enhancement_MIP.m      - Vessel enhancement and MIP export
4_vascular_quantification.m       - Vascular parameter quantification
5_anisotropy_index_visualization.m - Anisotropy index computation
6_depth_color_MIP.m               - Depth color-encoded projection
